## 📊 Women Literacy Prediction
ML dashboard classifying Indian districts by women's literacy using NFHS-5 data (706 districts, 109 socio-economic indicators). Compares Logistic Regression and Random Forest, with an interactive district explorer and live prediction tool.

**Key findings:**
- Random Forest outperforms Logistic Regression (82.4% vs. 72.5% accuracy)
- Malnutrition (Low BMI) is the strongest ML predictor of literacy
- Sanitation access shows the strongest infrastructure correlation (r ≈ 0.67)
- Child marriage is strongly negatively correlated with literacy (r ≈ −0.61)
- Health insurance coverage shows almost no correlation with literacy (r ≈ 0.01)

**Skills:** Scikit-learn · Feature Engineering · Feature Importance · Data Visualisation · Dashboard Development

## 🎓 Online vs. Offline Learning: A Statistical Analysis
*Undergraduate Research Thesis — B.Sc. (Hons.) Statistics, Banaras Hindu University*

This research investigates how learning environments (online vs. offline) influence students' academic experiences, examining the relationship between learning preferences, academic performance, stress levels, sleep patterns, and screen time using primary survey data from 110 undergraduate respondents.

**Methodology:** Data cleaning & validation, frequency/percentage analysis, pivot tables & charts, correlation analysis, Chi-Square test of independence, and multiple linear regression.

**Key finding:** Students show a stronger preference for offline learning, largely due to better classroom interaction and concentration. Stress and screen time show a stronger association with academic experience than sleep patterns.

**Future scope:** Extending the project with Python (Pandas, NumPy, Statsmodels, Scikit-learn) and an interactive Streamlit dashboard for predictive modelling and feature importance analysis.

**Skills:** Survey Design · Hypothesis Testing · Correlation & Regression · Chi-Square Testing · EDA · Data Visualization
